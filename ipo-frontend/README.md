# IPO Frontend

This is the frontend application for the IPO platform built with ReactJS and TailwindCSS.

## Features

- Upcoming IPO Web Page with search, filters, and detailed view.
- IPO Admin Dashboard with login, IPO management (Add/Edit/Delete).
- Fully responsive design for mobile, tablet, and desktop.
- ESLint and Prettier configured for code quality and formatting.

## Setup

1. Install dependencies:

```bash
npm install
```

2. Run the development server:

```bash
npm run dev
```

3. Build for production:

```bash
npm run build
```

## Folder Structure

- `src/components` - Reusable React components
- `src/pages` - Page components
- `src/utils` - Utility functions and constants
- `src/assets` - Images, icons, and other assets

## Styling

TailwindCSS is used for styling with utility-first approach.

## Notes

- Use environment variables for configuration.
- Follow coding standards and accessibility guidelines.
- Use Chrome DevTools for responsive testing.
